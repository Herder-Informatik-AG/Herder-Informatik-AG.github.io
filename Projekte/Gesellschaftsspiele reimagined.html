Gesellschaftsspiele reimagined

In diesem Projekt soll es darum gehen, wie man Gesellschaftsspiele mit informatischen Systemen unterstützen / verbessern kann um ein neues Spielerlebnis zu erschaffen oder das vorhandene Erlebnis zu verbessern.

Werwölfe von Düsterwald

Ein mögliches Beispiel wäre das Spiel "Die Werwölfe von Düsterwald". Dort könnte man beispielweise mehrere Funktionalitäten auf die Smartphones der Mitspieler auslagern, wodurch man zum spielen keine Karten mehr benötigen würde. Ein weiterer Schritt wäre die Digitalisierung des Spielleiters, sodass bei bedarf alle Spieler mitspielen können und das System von einem digitalen Gerät geleitet wird.

Als System für die Dorfabstimmung könnte dann beispielweise das Spiel "Among Us" als Vorbild dienen.

Skills

In diesem Projekt geht es in erster Linie um Webprogrammierung, da so kein Spieler eine App installieren muss und jede Person somit die Software nutzen kann. Dies bedeutet somit Programmierung in HTML und JavaScript (bzw. TypeScript).
